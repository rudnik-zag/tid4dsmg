    Configure and compile COLMAP:

    git clone https://github.com/colmap/colmap.git
    cd colmap
    mkdir build
    cd build
    cmake .. -GNinja
    ninja
    sudo ninja install

    colmap -h
    colmap gui

    Build PyCOLMAP:

        On Linux and macOS:

        python -m pip install ./pycolmap/

        Ako nevidi cublass dodati pre [project]:
